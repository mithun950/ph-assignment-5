Important FAQ

What is the Document Object Model?

The DOM (Document Object Model) is a programming interface that represents the structure of an HTML or XML document as a tree of objects. It allows programs like JavaScript to manipulate, access, and update the content, structure, and style of a web page dynamically. Each element, attribute, and text within the HTML is represented as a node in this tree, making it possible to interact with the page programmatically.

How do you select an element from the (DOM)?

To select an element from the DOM, you can use several JavaScript methods, depending on how you want to target the element (by id, class, tag name, or other selectors)

What is event delegation in the context of the DOM and why is it usefull?

Event delegation is a technique in JavaScript where a single event listener is added to a parent element to manage events for all its child elements, rather than adding separate event listeners to each child element. This works because of event bubbling, where events triggered on child elements bubble up through their parent elements

How do you multiple an elements attributes and style using the DOM?

To manipulate an element's attributes and style using the DOM, you can use JavaScript to directly access and modify the attributes and CSS properties of the element.
